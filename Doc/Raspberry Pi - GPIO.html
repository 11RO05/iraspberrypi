What is GPIO ?

GPIO = General-purpose input/output (GPIO)



Two different Mode for GPIO.

The GPIO.BOARD option specifies that you are referring to the pins by the number of the pin the the plug - i.e the numbers printed on the board (e.g. P1) and in the middle of the diagrams below.

The GPIO.BCM option means that you are referring to the pins by the "Broadcom SOC channel" number, these are the numbers after "GPIO" in the green rectangles around the outside of the below diagrams:



All the GPIO pins can be reconfigured to provide alternate functions, SPI, PWM, I²C and so.

1) GPIO control from shell
     
Export the required GPIO

echo “17” > /sys/class/gpio/export

Set the direction

echo “out” > /sys/class/gpio/gpio17/direction

Set the value

echo “1” > /sys/class/gpio/gpio17/value


Configure GPIO27 as input and read its value

1. Export the required GPIO

echo “27” > /sys/class/gpio/export

2. Set the direction

echo “in” > /sys/class/gpio/gpio27/direction

3. See the Set the value

cat /sys/class/gpio/gpio27/value

The code

Scratch has a BROADCAST function. You can create a BROADCAST message for the pins.

BROADCAST PIN11HIGH

or BROADCAST PIN11ON

So when you connect a function such as

WHEN SPACE BAR PRESSED, BROADCAST PIN11HIGH



The LED will light up. If it does not work, check your wiring. If you connect the wire from Pin11 to Pin1, this supplies 3.3V and the LED will light up.

To turn the LED off,

BROADCAST PIN11LOW

or BROADCAST PIN11OFF

You can also control Power by creating a VARIABLE

Create a VARIABLE called Power11.

You can then set the VARIABLE to a number between 0 and 100.

3) Python Library

#!/usr/bin/python
import RPi.GPIO as GPIO
GPIO.setmode(GPIO.BCM)
GPIO.cleanup()
GPIO.setwarnings(False)
GPIO.setup(17,GPIO.OUT)
GPIO.setup(27,GPIO.OUT)
print "Lights on"
GPIO.output(17,GPIO.HIGH)
GPIO.output(27,GPIO.HIGH)

#!/usr/bin/python
import RPi.GPIO as GPIO
GPIO.setmode(GPIO.BCM)
GPIO.cleanup()
GPIO.setwarnings(False)
GPIO.setup(17,GPIO.OUT)
GPIO.setup(27,GPIO.OUT)
print "Lights off"
GPIO.output(17,GPIO.LOW)
GPIO.output(27,GPIO.LOW)

  14.                         # different names for each port
